# **Dodge the Obstacle**

A simple C console game where you control a player (`^`) and dodge falling obstacles (`O`).
Your goal is to survive as long as possible while the speed increases over time.

## **How to Play**

* Use **LEFT** and **RIGHT arrow keys** to move between three lanes.
* Avoid the falling obstacles.
* You have **one shield** that protects you from a single collision.
* Score increases every time you dodge an obstacle.
* Game ends if you get hit without a shield.

## **Features**

* Progressive speed increase
* Single-use shield with random recharge
* Real-time score system
* Simple and fast console rendering
